# Requirement :

**Install Python3**

*For Ubuntu* :
```bash
	sudo apt install python3 python3-pip
```
# Run The code :

1. Clone the repository
```bash
	git clone https://github.com/sumony2j/Soft-Computing.git
```
2. In Command Line Type :
```bash
	python3 script_name.py
```
